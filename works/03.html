ポートフォリオサイト

XD / HTML / CSS / jQuery / Git

当ポートフォリオサイトです。
XDにてワイヤーフレーム作成からデザイン、コーディングまで、一から自分で制作しました。
バージョン管理ツールとしてGitを使用しております。
ベース、メイン、アクセントのカラー選定として、親しみやすく堅苦しさをあまり感じない印象になるような配色にしました。
Web関連の用語があまり得意でないという方もターゲットとし、できるだけ日本語を多めに、わかりやすさを心がけました。
全体的にシンプルにしていますが、PCならではのマウスオーバー時は少し動きをつける等こだわっています。
